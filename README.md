# jsonParser

a simple json parser that reads a json string and convert it to java objects (pojo).

Libraries used : jackson-core-2.8.3, jackson-annotations-2.8.3, jackson-databind-2.8.3

     |-Meta

     |-Data

           |-userName

           |-bio

           |-website

           |-profile_picture

           |-full_name

           |-Counts

                   |-media

                   |-followed_by

                   |-follows

           |-id
